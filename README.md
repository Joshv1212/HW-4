# Homework 4 - LLM API Experiment

Objective
This assignment uses Python code to interact with an LLM API instead of the web interface.

What I did
I used the OpenAI Python SDK and sent multiple prompts to the model using the Responses API.

Experiments
1. Basic explanation prompt
2. Same topic but for a different audience
3. Same topic with a formatting constraint

Observations
- The model changed its wording based on the intended audience.
- Adding formatting constraints made the response more structured.
- Simpler prompts gave broader answers, while more specific prompts gave more controlled outputs.
